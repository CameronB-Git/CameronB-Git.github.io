Reckitt eCommerce Site

                
                
                Reckitt eCommerce --> 
                

                    During my time at Ron Dearing UTC, one of my assignments within the Web Development Unit 
                    involved me creating a website using HTML, CSS, and Boostrap in order to transform Reckitt's, 
                    corporate and company based site, into a more eCommerce design. We first needed to design the website, in which I used
                    Adobe Xd, which helped me have an idea of how I wanted the website to look and work. 
                

                
                

                    I also ventured into the use of JavaScript, which wasn't required, but I felt that doing so would enhance my project
                    drastically; this meant that I could have full functionality, such as a Basket and Checkout System. 
                    Overall, this was a great project for me to work on, as it allowed me to gain a better understanding of how Markup and Styling Languages can 
                    be paired up with more functional programming languages to create a fully operative website.
